1. 원격저장소 로컬로 복제하기
   git clone https://github.com/sang105kr/product2

2. 인텔리제이 실행하기
   project2/build.gradle    마우스우클릭

3. 오라클서버 구동후 sqldeveloper에서 아래 스크립트 실행
	 admin 계정으로  product 계정생성      sql/admin.sql
	 product계정으로 product테이블 생성   sql/product.sql

4. 인텔리제이 환경설정
   프로젝트생성 및 인텔리제이환경설정.pdf

5. 웹APP실행
	Tomcat started on port(s): 9080 (http) with context path ''
	Started ProductApplication in 10.702 seconds (JVM running for 12.226)

6. 초기페이지 http요청
   http://localhost:9080
